# mvvm-sample-kotlin

# Architecture

MVVM + Clean Architecture

- Local : DB (Room) / Cache
- Remote : Retrofit
- Data : Repository / Data source
- Domain : Usecase
- Presentation : View / ViewModel

## Language
- Kotlin

## Library

- Android Jetpack
  * DataBinding
  * Navigation
  * Room
  * ViewModel
- Dagger2
- Gson
- Glide
- Retrofit2
- RxJava2 (RxKotlin)
