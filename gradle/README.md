# mvvm-sample

Android MVVM 패턴을 이용하여 구현된 Sample code 입니다.

- Kakao API를 이용하여 이미지 / 비디오의 Thumbnail 출력
- 원하는 Thumbnail을 선택하여 내 보관함에 추가 (DB에 저장)
- Kakao Developer (https://developers.kakao.com/) 에서 KAKAO_REST_API_KEY 등록 필요

## Language
- Java

## Library
- Data Binding
- Dagger2
- ViewModel (AAC)
- ViewPager
- RxJava2 (RxAndroid)
- Retrofit2
- Gson
- Picasso
- Room
